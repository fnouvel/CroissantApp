<!-- Backend auth endpoints created in Plan 01 -->

POST /api/auth/register
  Request: { "username": string, "password": string }
  Response 201: { "id": int, "username": string }
  Response 400: { "detail": "Username already taken" }

POST /api/auth/login
  Request: { "username": string, "password": string }
  Response 200: { "access_token": string, "token_type": "bearer", "user": { "id": int, "username": string } }
  Sets HttpOnly cookie: refresh_token (30 day expiry, samesite=lax)
  Response 401: { "detail": "Invalid credentials" }

POST /api/auth/refresh
  Reads refresh_token from HttpOnly cookie (browser sends automatically with credentials: "include")
  Response 200: { "access_token": string, "token_type": "bearer", "user": { "id": int, "username": string } }
  Response 401: { "detail": "No refresh token" | "Invalid refresh token" }

POST /api/auth/logout
  Deletes refresh_token cookie
  Response 200: { "message": "Logged out" }

All /api/bakeries/* and /api/bakeries/*/ratings endpoints:
  Require header: Authorization: Bearer {access_token}
  Response 401 if missing or invalid
